# Traffic Enforcement Impact Study

A data science project built in Python to analyze whether increased traffic enforcement correlated with lower traffic accident rates. This project uses data cleaning, visualization, correlation analysis, and predictive modeling to examine the relationship between traffic citations and accidents.

## Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Monthly and yearly aggregation
- Data visualization
- Correlation analysis
- Predictive modeling with linear regression
- Predictive modeling with decision tree regression

## Tech Stack
- Python
- pandas
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## Project Overview
This project explores whether higher levels of traffic enforcement are associated with changes in traffic accident rates. Using citation and accident data, the analysis focuses on identifying trends over time, visualizing relationships in the data, and testing whether predictive models can capture patterns between enforcement activity and accident outcomes.

The project includes data preparation, summary statistics, visual analysis, correlation analysis, and machine learning models to better understand the strength and nature of the relationship.

## How to Run
1. Clone the repository.
2. Open the project directory.
3. Install the required Python libraries if needed:
   `pip install pandas numpy matplotlib scikit-learn jupyter`
4. Launch Jupyter Notebook:
   `jupyter notebook`
5. Open the notebook file and run the cells in order.
